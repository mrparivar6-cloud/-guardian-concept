# AGI Guardian — Concept Registry (v0.1)

> **Canonical Name:** AGI Guardian  
> **Role:** AGI-based Observer, Listener, and Voice Advisor  
> **Scope:** Non-executive / Non-governing  
>
> This repository is a **legacy conceptual registry**.  
> The repository identifier `-guardian-concept` is preserved for historical continuity.


## Overview
Guardian is a conceptual framework for a personal artificial intelligence
system acting as a guardian, advisor, and spokesperson for its human user.

This repository exists solely as a **concept registry and public timestamp**.
It does not represent a product, service, or implementation.

## Core Roles (Conceptual)
- Decision awareness and advisory support
- Risk, safety, and boundary monitoring
- Voice-based interaction (speaking & listening)
- Representation and spokesperson assistance
- Human-in-the-loop by design

## Scope
Included:
- Concept definition
- Intent and role boundaries
- High-level functional description

Excluded:
- Source code
- Model training
- Deployment claims
- Commercial or financial promises
- AGI claims

## Non-Implementation Notice
Guardian is not autonomous and has no decision authority.
All actions and decisions remain under direct human control.

This repository contains **no executable system**.
